def unfold(input, kernel_size: int, stride: int)

1D only unfolding similar to the one from PyTorch. However PyTorch unfold is extremely slow.

Given an input tensor of size [*, T] this will return a tensor [*, F, K] with K the kernel size, and F the number of frames. The i-th frame is a view onto i * stride: i * stride + kernel_size. This will automatically pad the input to cover at least once all entries in input.

Args

input : Tensor
tensor for which to return the frames.
kernel_size : int
size of each frame.
stride : int
stride between each frame.

Shape

  • Inputs: input is [*, T]
  • Output: [*, F, kernel_size] with F = 1 + ceil((T - kernel_size) / stride)

Warning: unlike PyTorch unfold, this will pad the input

so that any position in input is covered by at least one frame.
